# Live-Weather-AQI-Insights-Dashboard
An interactive Power BI dashboard providing real-time weather analytics and air quality monitoring across multiple cities. This project leverages live API integration to deliver actionable environmental data with a focus on high-fidelity visualization and user-centric design.

Key Features: 7-day temperature forecasts, real-time AQI tracking, and dynamic weather-based backgrounds.

Tech Stack: Power BI Desktop, Power Query (M), DAX, and WeatherAPI.
